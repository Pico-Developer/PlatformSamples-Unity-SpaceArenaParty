</p>

Space Arena Party 是一个多人社交游戏 Demo，集成了 PICO 平台服务中的 “好友” 服务、“社交互动” 服务以及 “房间&匹配” 服务。你可以在 Demo 中体验以下能力：
- Avatar 和角色移动
- 创建和加入虚拟房间
- 实时多人互动
- 好友系统，包括查看好友列表、邀请好友
